When we want to change the background color of the button we change its "background-color" property whereas when changing the text color we change the property called"color"

To round the edges of our button, we will use the property "border-radius" and more are the number of pixels in border radius more rounded is the button. To change the cursor to a hand pointing, we will change the property "cursor" to value pointer.

Now, to specify to the language that following css styles is for this specific button, we are gonna use an attribute of html called the "class" attribute to which we assign a name and then the given button is identified by the name given in the class. to mention a class, we generally put a period and write class name after that

New things we used in style tag of css,
1.to set the colour of the border we used "border-color".
2.to set the border style, we used "border-style".
3. to set the border width we use "border-width".
4.to have space after the subscribe button we used the css property called "margin-right" and assigned it a value in pixels.

To edit the font in the tweet button:
1.We use property called "font-weight" to make the font bold.
